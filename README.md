# YouTube Creator Analytics

This project analyzes a real-world dataset of top YouTube creators to uncover insights related to subscriber growth, content categories, earnings, and geographic distribution.

## Objectives
- Identify top YouTube channels based on subscribers
- Analyze category-wise and country-wise trends
- Study earnings distribution and outliers
- Explore growth trends using recent subscriber data

## Dataset
The dataset contains information on YouTube creators including subscribers, video views, uploads, earnings, country-level indicators, and channel creation details.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Spyder

## Key Learnings
- Strong correlation between subscribers and video views
- Earnings vary significantly across categories
- Creator success is influenced by category, geography, and longevity
